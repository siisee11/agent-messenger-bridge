문서

개요 시작하기 CLI 레퍼런스 Discord 봇 설정 Slack 봇 설정 릴리즈 가이드

CLI 레퍼런스

명령어는 전역 명령어(어디서나 실행 가능)와 프로젝트 명령어(프로젝트 디렉토리에서 실행)로 나뉩니다.

전역 명령어

명령어 용도 예시
discode onboard 최초 설정: 토큰, 서버 감지, 기본 CLI, 권한 모드 설정. discode onboard --token YOUR_BOT_TOKEN
discode daemon <start|stop|status> 전역 데몬 프로세스 제어. discode daemon status
discode list 등록된 프로젝트 전체 목록 조회. discode list
discode agents 감지된 AI CLI 바이너리 확인. discode agents
discode tui 인터랙티브 터미널 UI 실행. discode tui
discode config [options] 전역 설정 값 조회/수정. discode config --show

프로젝트 명령어

명령어 용도 예시
discode new [agent] [options] 설정, 데몬 시작, 채널 생성, tmux 연결까지 빠르게 시작. discode new claude
discode start [options] 등록된 프로젝트 시작. discode start -p my-app --attach
discode stop [project] 프로젝트 중지, tmux 세션 정리, 상태 제거. discode stop --keep-channel
discode status 프로젝트 상태 조회. discode status
discode attach [project] 현재 또는 지정된 프로젝트의 tmux 세션에 연결. discode attach my-app

자주 쓰는 흐름

# 최초 1회 설정
discode onboard

# 새 프로젝트 세션 시작
cd ~/projects/my-app
discode new

# 상태 점검
discode daemon status
discode status

에이전트 바이너리는 command -v <binary>로 감지합니다. 감지되지 않으면 CLI를 먼저 설치하고 discode agents를 다시 실행하세요.
