Slack 봇 설정

Slack 앱을 한 번 생성하고 필요한 토큰을 발급하면 Discode를 Slack에 연결할 수 있습니다. 아래 단계는 docs/SLACK_SETUP.ko.md를 바탕으로 정리했습니다.

1) Slack 앱 생성

  1. Slack App 관리 페이지에 접속합니다.
  2. Create New App - From scratch를 선택합니다.
  3. 앱 이름(예: Discode Bot)과 워크스페이스를 선택합니다.

2) Socket Mode 활성화 및 앱 토큰 발급

  1. 사이드바의 Socket Mode에서 기능을 활성화합니다.
  2. connections:write 권한으로 App-Level Token을 생성합니다.
  3. xapp-... 토큰을 복사해 안전하게 보관합니다.

3) Bot Token Scope 설정

OAuth & Permissions - Bot Token Scopes에서 다음을 추가하세요:

Scope 필요한 이유
chat:write 봇 메시지 전송.
channels:history 공개 채널 메시지 읽기.
channels:read 채널 목록 조회.
channels:manage 프로젝트 채널 생성 및 아카이브.
groups:read 비공개 채널 메타데이터 조회.
files:read / files:write 파일 읽기 및 업로드.
reactions:read / reactions:write 승인 리액션 처리.

4) 이벤트 활성화 및 워크스페이스 설치

  1. Event Subscriptions를 켜고 message.channels 이벤트를 추가합니다.
  2. Install App에서 Install to Workspace를 클릭합니다.
  3. xoxb-... Bot User OAuth Token을 복사합니다.

Socket Mode를 사용하므로 Event Subscriptions의 Request URL은 필요하지 않습니다.

5) 온보딩 실행 및 확인

discode onboard --platform slack
discode config --show
discode new

비대화형 환경에서는 토큰을 직접 전달할 수 있습니다:

discode config --platform slack \
  --slack-bot-token xoxb-your-bot-token \
  --slack-app-token xapp-your-app-token

xoxb-...와 xapp-... 토큰은 절대 커밋하거나 외부에 공유하지 마세요.

빠른 참조 카드

1. api.slack.com/apps 에서 앱 생성
2. Socket Mode 활성화 + xapp 토큰 발급
3. Bot Token Scopes 추가
4. Event Subscriptions: message.channels 추가
5. 앱 설치 후 xoxb 토큰 복사
6. discode onboard --platform slack 실행
7. discode new 시작
