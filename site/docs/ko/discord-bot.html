Discord 봇 설정

Discode에서 사용할 봇을 한 번만 생성하고 서버에 초대하면 됩니다. 아래 단계는 docs/DISCORD_SETUP.ko.md를 요약했습니다.

1) 봇 애플리케이션 생성

  1. Discord Developer Portal 에 접속합니다.
  2. 새 애플리케이션을 만든 뒤 Bot 탭으로 이동합니다.
  3. 봇 토큰을 복사해 안전하게 보관합니다.
  4. 토큰은 공개 저장소나 채팅에 절대 공유하지 않습니다.

2) 인텐트 활성화

Privileged Gateway Intents에서 다음을 활성화하세요:

  • MESSAGE CONTENT INTENT (필수)
  • SERVER MEMBERS INTENT (선택)

Discode의 승인 플로우는 리액션 이벤트(비특권 인텐트)도 사용합니다.

3) 서버에 봇 초대

  1. OAuth2 -> URL Generator로 이동합니다.
  2. Scope에서 bot을 선택합니다.
  3. 아래 권한을 체크합니다.
  4. 생성된 URL을 열어 대상 서버에 봇을 초대합니다.
권한 필요한 이유
View Channels 프로젝트 채널 접근 및 추적.
Manage Channels 프로젝트별 채널 자동 생성.
Send Messages 에이전트 출력과 상태 메시지 전송.
Send Messages in Threads 스레드 기반 대화 지원.
Embed Links 인터랙티브 질문/승인 임베드 표시.
Read Message History 컨텍스트 추적 및 리액션 처리.
Add Reactions Discord 리액션 승인 플로우.

4) 온보딩 실행 및 확인

discode onboard
discode config --show
discode new

서버 자동 감지에 실패하면 봇이 올바른 길드에 초대되었는지, 필요한 권한이 있는지 확인하세요.

빠른 참조 카드

1. discord.com/developers/applications 에서 앱+봇 생성
2. MESSAGE CONTENT INTENT 활성화
3. Bot 탭에서 토큰 복사
4. OAuth2 > URL Generator: scope=bot
5. 권한 선택 후 서버에 초대
6. discode onboard 실행
7. discode new 시작
