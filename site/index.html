What is Discode?

Discode runs your AI coding CLI in tmux and relays progress to Discord in real time.

  • - Multi-agent Works with Claude Code, Gemini CLI, and OpenCode
  • - Project isolation Each project gets a dedicated Discord channel
  • - Persistent sessions tmux sessions survive disconnects and restarts
  • - Single daemon One bot connection manages multiple projects
  • - Relay-only No wrappers, no cloud dependency, no hidden execution layer
  • - Realtime stream Relays agent hook events to Discord in real time
