문서

개요 시작하기 CLI 레퍼런스 Discord 봇 설정 Slack 봇 설정 릴리즈 가이드

시작하기

Discode는 로컬 AI 코딩 CLI를 Discord 채널에 연결합니다. 사전 요구 사항을 확인한 뒤 온보딩과 discode new로 시작하세요.

사전 요구 사항

  • Bun 1.3+ 및 tmux 3.0+.
  • 봇이 초대된 Discord 서버와 봇 토큰.
  • 지원되는 AI CLI 중 하나 이상: claude, gemini, opencode.

아직 봇 설정을 하지 않았다면 Discord 봇 설정을 먼저 진행하세요.

Slack을 사용한다면 Slack 봇 설정에서 xoxb-..., xapp-... 토큰 발급 절차를 먼저 진행하세요.

Discode 설치

다음 중 하나를 선택하세요:

curl -fsSL https://discode.chat/install | bash
npm install -g @siisee11/discode
bun add -g @siisee11/discode
git clone https://github.com/siisee11/discode.git
cd discode
bun install
bun run build
bun link

온보딩 1회 실행

onboard는 토큰 저장, 서버 ID 감지, 기본 에이전트 설정을 한 번에 처리합니다. 

discode onboard
discode config --show
discode config --server SERVER_ID

최초 설정은 onboard를 사용하세요. 이후 값 수정은 config 명령어로 관리하면 됩니다.

첫 세션 시작

cd ~/projects/my-app
discode new

new는 에이전트 감지, Discord 채널 생성, 데몬 시작, tmux 실행/연결까지 자동으로 진행합니다. 

discode new claude
discode new --no-attach

다음 단계

  • CLI 레퍼런스에서 전체 명령어를 확인하세요.
  • discode status, discode list로 상태를 점검하세요.
  • 에이전트를 멈추지 않고 tmux를 분리하려면 Ctrl-b d를 사용하세요.
