문서

로컬에서 AI 코딩 세션을 실행하고 Discord에서 진행을 확인하세요.

Discode는 AI 코딩 CLI를 tmux에서 실행하고, 출력을 캡처해 Discord 전용 채널로 릴레이합니다. 이 문서는 프로젝트 README와 docs/ 가이드를 바탕으로 구성되었습니다.

시작하기 -> Discord 봇 설정

시작하기

Discode를 설치하고 온보딩을 진행한 뒤 첫 프로젝트 세션을 시작하세요.

Slack 봇 설정

Slack 앱 생성, 토큰 발급, 온보딩 연결까지 단계별로 안내합니다.

CLI 레퍼런스

전역 명령어와 프로젝트 명령어를 한 번에 확인하세요.

Discord 봇 설정

봇 생성, 인텐트 활성화, 권한 설정, 서버 초대까지 단계별로 안내합니다.

릴리즈 가이드

유지보수자를 위한 npm 배포 흐름과 검증 절차를 정리했습니다.

빠른 시작

프로젝트 디렉토리에서 보통 다음 순서로 시작합니다:

discode onboard
cd ~/projects/my-app
discode new

핵심 개념

로컬 우선 실행, 릴레이 전용 아키텍처, 영속 tmux 세션, 단일 데몬 기반 멀티 프로젝트 관리를 핵심으로 합니다.

원문 문서는 README.ko.md, DISCORD_SETUP.ko.md, RELEASE_NPM.ko.md 를 참고하세요.
